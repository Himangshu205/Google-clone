# Google-clone:-
# This is an attempt to make a clone of the google homepage
This project is a simple clone of the Google homepage UI, created using HTML and CSS.

Technologies that has been used in this:-
(i)HTML5
(ii)CSS3
(iii)Font Awesome (for icons)

## Features:-
-Google-style navigation bar
-Search bar with icons
-Language options section
-Footer layout similar to Google
-Responsive layout (basic)
