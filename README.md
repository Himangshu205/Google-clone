# Google-clone:-
# This is an attempt to make a clone of the google homepage
This project is a simple clone of the Google homepage UI, created using HTML and CSS.
The goal of this project was to practice layout design, positioning, and styling by recreating a familiar interface.

Technologies Used:-
(i)HTML5
(ii)CSS3
(iii)Font Awesome (for icons)

Features:
(i)Google-style navigation bar
(ii)Search bar with icons
(iii)Language options section
(iv)Footer layout similar to Google
(v)Responsive layout (basic)
